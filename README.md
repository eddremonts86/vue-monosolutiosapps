# VueJS & Vuetify
This is a complete app to start our vuejs projects as soon we clone it.
  - Elements 
    - Vue js 
    - Vuetify
    - Route
    - Axios
    - Store 
    - SASS / SCSS
    - Sweet Alert 2
    - Material Design Icons 
    - Eslint
    - Prettier
 
  - Test tools
  
    - Jest
    - Cypress
    
        
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Run your end-to-end tests
```
npm run test:e2e
```

### Lints and fixes files
```
npm run lint
```
